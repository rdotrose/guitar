Raw Lyrics Text

Underneath "Notes" and "Intro", paste and format the lyrics for the song. The convert button will create HTML code for the next section. A single return will create a line break. Two returns (a line break followed by an empty line) will mark the end of a paragraph.

Tab Copy (Preview and Adjust)

Use the menu bar at the bottom of the page place chords within the lyrics of your song. Type in the 'Chord Text' box or click one of the defined chords to specify which chord will be inserted. Click or touch a place in the lyrics to insert a chord. Chords can also be dragged and dropped within the lyrics section.
